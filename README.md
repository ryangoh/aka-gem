# aka - The Missing Alias Manager

aka generate/edit/destroy/find permanent aliases with a single command.

aka2 requires ruby and is built for bash and zsh users.

## Installation

    $ gem install aka2
    $ aka setup2

## Usage

    $ aka generate hello="echo helloworld"
    $ aka destroy hello
    $ aka edit hello
    $ aka find hello
    $ aka usage
    $ aka
    $ aka help

## Requirement

Ruby
